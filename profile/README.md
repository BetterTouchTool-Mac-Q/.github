## Overview

BetterTouchTool Advanced is a powerful input customization and automation application from folivora.ai that enables custom gestures, keyboard shortcuts, mouse actions, Touch Bar widgets, and window snapping, making every input device on your system dramatically more capable. It bridges the gap between the default input behavior that ships with the operating system and the personalized interaction model that power users need to work efficiently. The depth of customization available means that no two installations look alike — every user creates an input environment tailored to their specific workflow.

Custom trackpad and mouse gestures create new interactions — three-finger swipes, corner clicks, pressure-sensitive taps, pinch variations — that trigger any action from application switching through script execution. This extends the input vocabulary of your pointing devices far beyond their default capabilities, turning simple hardware into a sophisticated control surface. Force Touch gestures on supported trackpads add another dimension of input that most applications leave entirely untapped.

Window snapping and management provides keyboard and gesture-driven window positioning that tiles, resizes, and arranges windows with the precision and speed that the default macOS window management does not offer. Custom snap areas can be defined for any screen configuration, enabling layouts that match your specific monitor arrangement and workflow needs. Window management presets save entire multi-window arrangements and restore them with a single gesture or shortcut.

Touch Bar customization replaces the default Touch Bar with fully custom widgets, buttons, sliders, and information displays tailored to your workflow, making the Touch Bar genuinely useful rather than a row of rarely-used default controls. Widgets can display live data — system stats, weather, calendar events, script output — turning the Touch Bar into an active information display. Each application can have its own Touch Bar configuration, presenting context-relevant controls that change automatically as you switch between tasks.

For power users who want every input device — trackpad, keyboard, mouse, Touch Bar — to behave exactly the way they want rather than the way Apple designed it, BetterTouchTool Advanced is the customization platform that adapts the system to the user. Its breadth of input device support and action depth make it one of the most transformative utilities available for anyone who interacts with their computer through physical input devices.

## Performance

BetterTouchTool balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
